Mau Gratis?

Addons ini bisa anda peroleh secara gratis jika mengikuti Training Odoo di vitraining.com atau membeli E-BOOK "PANDUAN LENGKAP IMPLEMENTASI ODOO V10 UNTUK PERUSAHAAN RETAIL DI INDONESIA" di shop.vitraining.com maupun di Google Play.

Installation

  • install this addon on the database
  • download the proxy script from this LINK
  • install apache+php or nginx+php on the local computer and extract to above file info the htdocs folder
  • print Invoice, SO, PO directly to local dotmatrix printer
